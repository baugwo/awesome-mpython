
## Library Management

按以下library分类,进行归类:

| 目录           | 说明                            |
| :------------- | :------------------------------ |
| audio          | 音频相关                        |
| communications | 通讯相关                        |
| display        | 显示相关,如液晶显示屏驱动、灯带     |
| io             | IO 相关                        |
| iot            | 物联网相关                      |
| AI             | 人工智能相关                    |
| motion         | 动作类的驱动                    |
| sensors        | 传感器类的驱动                  |
